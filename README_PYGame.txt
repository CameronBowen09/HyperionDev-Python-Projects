PYGame: Capstone II

This is a game made with the pygame plug in in visual studio code.
It is a game where there are three ememies that come at the player
form the righthand side of the screen, if the player happens to 
touch one of the enemies the game will end and the console will
display you lose. There is a another object that comes towards the
player from the right, however this is a reward that the player
must try touch whist avoiding the emenies to win the gaem, if the
player touches the reward the console will display you win.

The key binding in the game are set to the up,down,left,right keys
to make the player move in the coresponding direction of the key 
they are pressing.
